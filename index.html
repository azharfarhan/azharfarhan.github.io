PENDIDIKAN

Pendidikan saya..... saya sekolah TK di TK Ar-Rahmah karena TK tersebut dekat dengan rumah saya. Setelah itu, saya melanjtkan SD Cijantung 07 Pagi. Setelah itu saya mendaftar ke salah satu sekolah favorit yang berada di Jakarta Timur yaitu SMPN 103 Jakarta Timur. Setelah lulus dari SMP 103, saya mencoba daftar di SMK-SMAK Bogor, tetapi untuk masuk sekolah favorit harus melakukan test terlebih dahulu. Dan saya mengikuti test tertulis tersebut walaupun saya tidak keterima lewat jalur rapot tetapi saya lolos lewat test tertulis. Saya sangat senang saat melihat pengumumman terdapat nomor test tertulis saya dan itupun berkat doa dari kedua orang tua saya. Saya senang sekali menjadi siswa SMK-SMAKBogor. Dan sekarang saya telah menempuh kelas 11

HOBI

Menurut saya semua orang pasti mempunyai hobi. Disini saya akan menceritakan sedikit tentang hobi saya. Saya sangat menyukai Hobi hiking (mendaki gunung), menurut saya dengan mendaki gunung saya dapat lebih munsyukuri dan menjaga kelestarian alam sekitar. dengan hiking ini saya juga mendapatkan banyak cerita dan pengalaman. Lalu ketika saya mendaki gunung saya menjadi mempunyai banyak kenalan dan saling berbagi cerita bersama waktu naik. Dan hobi saya yang kedua adalah bermain band bersama teman, tetapi hobi saya ini dilakukan ketika waktu smp. Dengan bermain band saya juga mempunyai banyak teman dan menjadi terkenal di smp kala itu. Dan band favorit saya waktu itu adalah Sheila On 7. Menurut saya band ini hampir tidak mempunyai "haters" karena lagu lagunya yang selalu meledak di pasaran dan enak untuk didengar.
